팀로그는 서버와 네트워크를 기반으로 교육하는 선린인터넷고등학교의 전문동아리 입니다. C언어의 바탕으로 서버를 다루는데 필요한 서버 사이드 언어를 공부하며 네트워크의 동작원리와 서버를 구축하고 유지하는데 필요한 여러 정보를 공유하는 등의 학생들 사이에 교류하는 사이로 이루어져있습니다.

교육 일정

서버

서버는 노드를 뒤짚어 엎어 하드를 막 뒤죽박죽 하는것을 말합니다.
본격적인 서버 수업은 학기 초 C언어로 프로그래밍 지식의 기본을 갖추고 교내서버구축대회를 대비하며 이러쿵저러쿵 하며 교육이 이루어질것입니다.

/* 수정 필요 */

네트워크

네트워크는 통신기기 이러쿵 저러쿵을 이용해 휘리릭 하는것을 말합니다.
네트워크 수업은 서버수업이 이루어지는 중 부수적으로 교육되며 본격적인 수업은 서버구축대회가 끝난 후가 될것입니다.

C언어

프로그래밍의 전반적인 기초를 다루기 위해 학기 초에는 C언어를 중점적으로 교육합니다.

웹 프런트

웹 서버를 다룰 때 화면에 보여질 프런트를 부수적으로 함께 배웁니다.
